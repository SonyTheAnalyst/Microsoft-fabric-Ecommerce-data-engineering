# Microsoft-fabric-Ecommerce-data-engineering

This project builds a complete Customer 360° data platform for an e‑commerce company using Microsoft Fabric and modern Azure Data Engineering practices.
The goal is to consolidate fragmented customer data from multiple operational systems into a single, analytics‑ready view that supports personalization, lifecycle analysis, and data‑driven decision‑making.

💼 Business Use Case



E‑commerce companies often operate across multiple channels:
• 	Website
• 	Mobile app
• 	Customer service
• 	Order management system


However, customer data is typically scattered across these systems, making it difficult to:
• 	Understand the full customer lifecycle
• 	Identify high‑value customers
• 	Personalize marketing campaigns
• 	Improve customer retention
• 	Analyze purchase and engagement behavior


The leadership team wants a single, unified Customer 360 view that integrates:
• 	Profile information
• 	Purchase behavior
• 	Payment history
• 	Customer support interactions
• 	Web/app activity
